# shhh

A quiet dark theme for [Zed](https://zed.dev). Built with AI, fine tuned for Dart syntax. WCAG AA tested

![shhh-preview](preview.png)

## Palette

| Token | Hex |
|-------|-----|
| Types & functions | `#6ca880` |
| Keywords | `#9880c0` |
| Strings | `#c89868` |
| Numbers & booleans | `#7da8d2` |
| Methods | `#b87880` |
| Variables | `#c0c0cc` |
| Params & props | `#a0a0ac` |
| Background | `#202024` |

## Install

Install via Zed extensions.
